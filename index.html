Estamos emocionados em apresentar o nosso robô em forma de carrinho, uma criação incrível que combina a tecnologia mais avançada com uma funcionalidade única: uma arma de agulha que pode ser ativada para ação rápida! Com este texto, pretendemos compartilhar informações essenciais sobre o nosso projeto, abrangendo tanto os consumidores quanto os investidores, de forma clara e envolvente.

LER MAIS O Robô

ARMAS

O Projeto da Arma:

O Projeto é Feito com a estrutura de um guarda chuva. Existe um sensor na parte de trás do robô que quando acionado, faz 2 polias girarem e faz a arma subir junto com agulhas afiadas para estourar o balão do oponente

  • Arma:22cm
  • 2 Motores, 2 polias e uma correia dentada
  • Agulhas afiadas

    • Hardware

    O Projeto do Hardware

    O Hardware Do robo é composto por:

  • Arduino Mega
  • Soquete de pilha
  • Dois Motores 5v para as rodas
  • Dois Motores 5v para as armas
  • Chave fim de Curso (sensor)
  • Shield de Motor
  • Shield Bluetooth

    RoBoCup

    A competição Robocup

    A Robocup é a competição mais antiga da FIAP. Alunos dos 1°s anos de Sistemas de Informação constroem robôs para participar de duelos, onde o principal objetivo é estourar o balão que o adversário carrega.
